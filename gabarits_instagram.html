📖 Comment utiliser ces gabarits

1
Ouvrir ce fichier dans Chrome (double-clic sur gabarits_instagram.html)
2
Choisir le template souhaité dans les onglets ci-dessus
3
Capture d'écran Windows : Win+Shift+S → sélectionner exactement le cadre du template
4
Ou clic-droit sur le template → Enregistrer l'image sous (si le navigateur le propose)
5
Uploader directement sur Instagram depuis le téléphone (AirDrop ou Google Drive)

📐 Formats Instagram expliqués

1:1
Carré 1080×1080px — format universel, visible en entier dans la grille du profil. Recommandé pour : bienvenue, kermesse, Club IA.
4:5
Portrait 1080×1350px — occupe plus de place dans le feed (meilleur engagement). Recommandé pour : inscriptions, bulletins, citations.
⚠
Éviter le 16:9 (format paysage) — Instagram recadre automatiquement et coupe les bords. Toujours utiliser 1:1 ou 4:5.
